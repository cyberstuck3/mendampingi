## About

**Mendampingi** is a web application designed to help new fathers:
- Screen for postpartum depression in their wives using a validated questionnaire (EPDS)
- Track their partner's emotional well-being over time
- Access targeted guidance and resources during the postpartum period
- Get AI-powered support to better understand what their family is going through

This project was built during **Summer Club Explorer** by Generation Girl × Merck Life Science Indonesia, as part of a session on AI ethics and responsible product development.

---

## Features

- 📋 **PPD Screening** — Edinburgh Postnatal Depression Scale (EPDS) questionnaire
- 📈 **Mood Tracker** — Log and monitor emotional well-being over time
- 💡 **Guidance & Resources** — Curated tips and information for new fathers
- 🤖 **AI Support** — Powered by Gemini via Google AI Studio
- 📱 **Fully Responsive** — Works well on both mobile and desktop

---

## Tech Stack

- **Frontend:** HTML, CSS, TypeScript, Vite
- **AI:** Gemini API via Google AI Studio
- **Deployment:** Google Cloud Run

---

## Run Locally

**Prerequisites:** Node.js

1. Clone the repository 
2. Install dependencies
3. Set up your environment — create a `.env.local` file and add your Gemini API key
   You can get a free API key at [Google AI Studio](https://aistudio.google.com)
4. Run the app
